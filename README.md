# Implementation of query recovery attacks

This repo contains implementation of several query recovery attacks on conjunctive/boolean queries addressed in:
"Efficient Query Recovery Attack on Boolean Queries"

The code is generally based on the implementation of CKWS-attack by Dijkslag et al., available at https://github.com/marcowindt/passive-ckws-attack.
